Project description

AcS(USA Accidents Smart Visualizer) is a Web tool for visualizing data about accidents in USA between 2016-2020.

Structure

The projects consists of managing and displaying data in different types of visual formats (pie charts, line charts, table, interactive map).

Technologies

Python

Python is used for processing the data from the given dataset (a .csv file).

Sqlite

A Sqlite database is used for storing and extracting data.

JavaScript

JavaScript is used for creating charts (using Google Charts), and establishing connection and communication between the application and the database.

HTML & CSS

HTML & CSS are used for creating the interface of the application.

Heroku

Heroku is a service that can host our application on cloud, deploying the latest version we have on our git repository.

Functionality

Visualize data about accidents in USA between 2016-2020

The user can interact with the elements by selecting a data subcategory

Data about accidents regarding:

  • City
  • State
  • Weather Condition
  • Date
  • Severity
  • Street

The user can export data

Data can be exported as :

  • SVG
  • XML
  • CSV

The application consists of four pages. Browsing between them can be done using the navigation bar.

User guide

Hover over the map to see number of accidents in every state.

Hover over the pie charts to see the number of accidents in two states comparatively.

Hover over the line chart to see the evolution of the number of accidents in the defined period.

In the navigation bar, click 'Export' to be redirected to the exports page. Choose the desired format.

In the navigation bar, click 'Report' to be redirected to the report page.

In the navigation bar, click 'Contact' to be redirected to the contact page.

In the navigation bar, click 'Home' to be redirected to the home page.
