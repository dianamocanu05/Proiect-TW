1. Descrierea proiectului

AcS (USA Accidents Smart Visualizer) este o aplicație web care permite utilizatorilor să vizualizeze date despre accidentele din USA din perioada 2016-2020.

2. Introducere

Proiectul este o aplicație web ce constă în gestionarea și afișarea datelor în diferite tipuri de formate (diferite diagrame, tabel, hartă interactivă).

3. Structură

3.1 Tehnologii utilizate

3.1.1 HTML & CSS

Pentru a crea interfața aplicației am folosit HTML și CSS.

3.1.2 Javascript

JavaScript este utilizat pentru a crea diagrame (folosind Google Charts) și pentru a stabili conexiunea și comunicarea între aplicație și baza de date.

3.1.3 Sqlite

Pentru stocarea și extragerea datelor este utilizată o bază de date Sqlite.

3.2 Pagini

Site-ul este compus din 4 pagini, iar navigarea între ele se poate face folosind bara de navigare.

  • Home
  • Report
  • Export
  • Contact

4. Funcționalitate

  1. Utilizatorul își alege tipul de date pe care dorește să îl vizualizeze și modul de vizualizare. În funcție de alegerile făcute, i se vor afișa datele cerute sau va primi o eroare deoarece datele introduse nu sunt corecte.
  2. De asemenea, utilizatorul poate salva statisticile și vizualizările într-un format ales de acesta (CSV, WebP, SVG)
